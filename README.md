## 酷欧天气 coolweather 
《Android第一行代码》酷欧天气案例
酷欧天气是一款基于Android端开源的天气预报软件，具备查看全国的省市县、查询任意城市天气、自由切换城市、手动更新天气、后台自动更新天气等功能。酷欧天气中的天气数据由和风天气提供，背景图片由必应提供，代码遵循Apache v2 License开源协议。
